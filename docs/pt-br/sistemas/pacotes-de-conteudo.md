# Pacotes De Conteudo

Sistemas podem distribuir conteudo inicial como atores, itens, diarios, cenas e outros recursos suportados.

## Uso

Declare packs no manifest do sistema. O importador valida paths, formatos e referencias antes de inserir conteudo na campanha.

## Regras

- Conteudo deve ficar dentro do diretorio do sistema.
- Assets precisam ser referenciados por paths seguros.
- Packs devem ser pequenos o bastante para revisar e testar.
